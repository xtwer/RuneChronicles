## 🎯 完整游戏流程

```
GameBootstrap (启动)
    ↓
MainMenuUI (主菜单)
    ↓ [开始游戏]
CharacterSelectUI (角色选择)
    ↓ [选择法师/战士]
MapUI (地图)
    ├─ [战斗节点] → BattleUI → BattleResultUI → RewardUI → MapUI
    ├─ [精英节点] → BattleUI → BattleResultUI → RewardUI → MapUI
    ├─ [BOSS节点] → BattleUI → BattleResultUI → RewardUI → MapUI
    ├─ [商店节点] → ShopUI → MapUI
    ├─ [宝箱节点] → (获得金币) → MapUI
    └─ [融合按钮] → FusionUI → MapUI
```

### 完整通关路径
1. **主菜单** → 开始游戏
2. **选择角色** → 法师或战士
3. **第1-4层** → 战斗/商店/宝箱
4. **第5层** → BOSS战（第一关）
5. **第6-9层** → 战斗/商店/宝箱
6. **第10层** → BOSS战（第二关）
7. **第11-14层** → 战斗/商店/宝箱
8. **第15层** → 最终BOSS战
9. **通关！** 🎉

## 🎮 游戏特色功能

### 1. 卡牌拖拽系统
- **IBeginDragHandler/IDragHandler/IEndDragHandler**
- 拖拽卡牌到敌人上打出
- 半透明拖拽效果
- 自动返回原位

### 2. 商店系统
- **随机5张卡牌** - 每张50金币
- **随机2个遗物** - 每个150金币
- **移除卡牌** - 75金币/次
- **实时金币显示**
- **购买后刷新**

### 3. 融合系统
- **选择两张卡牌**
- **预览融合结果**
- **消耗融合点**（每次融合消耗，战斗获得）
- **自动添加到牌库**
- **移除原卡牌**
- **10种预设配方** + 随机融合

### 4. 战斗系统
- **回合制** - 玩家回合 → 敌人回合
- **能量系统** - 每回合3点能量
- **护盾机制** - 回合结束清除
- **抽牌系统** - 回合开始抽5张
- **敌人AI** - 3种行为模式

### 5. 地图系统
- **15层渐进**
- **5种节点类型**（战斗/精英/BOSS/商店/宝箱）
- **每层3条路径**
- **BOSS层固定**（5/10/15层）
- **融合入口**（随时可进入）

## 🚀 如何启动游戏

### 方法1：使用GameBootstrap（推荐）

1. **打开Unity项目**
   ```bash
   cd ~/Desktop/RuneChronicles/RuneChronicles
   ```

2. **创建启动器**
   - Hierarchy → Create Empty
   - 重命名为 `GameBootstrap`
   - Add Component → `GameBootstrap`

3. **点击Play** ▶
   - 自动初始化所有系统
   - 自动启动主菜单
   - 开始游戏！

参考 `HOW_TO_PLAY.md` 获取详细启动指南。

## 💡 游玩建议

### 新手攻略
1. **先选法师** - 玩法简单，输出高
2. **优先升级攻击卡** - 快速结束战斗
3. **前期避开精英** - 血量不够容易死
4. **善用护盾** - 护盾回合结束才清除
5. **商店优先买遗物** - 遗物永久生效

### 进阶策略
1. **平衡卡组** - 攻击30% + 防御40% + 能力30%
2. **关注卡牌费用** - 低费卡更灵活
3. **融合创造神卡** - 传说卡非常强
4. **利用敌人意图** - 看敌人下回合做什么
5. **金币管理** - 优先买遗物，次优买卡

### 高级技巧
1. **融合时机** - 有融合点就融合
2. **删卡策略** - 删除低价值卡牌
3. **路径选择** - 平衡战斗和商店
4. **BOSS准备** - BOSS前确保卡组强度
5. **遗物组合** - 某些遗物有协同效果

## 🚀 如何启动游戏（只需3步）

### 方法1：使用GameBootstrap（推荐）

1. **在Unity中打开项目**
   ```bash
   cd ~/Desktop/RuneChronicles/RuneChronicles
   # Unity Hub → 打开项目
   ```

2. **创建启动器**
   - Hierarchy窗口右键 → `Create Empty`
   - 重命名为 `GameBootstrap`
   - Inspector → `Add Component` → 搜索 `GameBootstrap`
   - 添加组件

3. **点击Play按钮** ▶
   - 自动初始化所有系统
   - 自动启动主菜单
   - 开始游戏！

### 方法2：手动创建主菜单

如果方法1不工作：

1. 创建空物体 `MainMenuUI`
2. 添加 `MainMenuUI` 组件
3. 确保场景中有 `EventSystem`
4. 点击Play

## 🎯 完整游戏流程

### 1. 主菜单
- **开始游戏** → 角色选择
- **设置** （未实装）
- **退出游戏**

### 2. 角色选择
- **符文法师** - 生命80，魔法输出
- **符文战士** - 生命100，反击防御
- 点击角色卡片即可选择

### 3. 地图探索
- 每层3个节点可选
- 节点类型：
  - 🟢 **战斗** - 普通敌人（40HP）
  - 🔴 **精英** - 强力敌人（60HP）
  - 🔥 **BOSS** - 超强敌人（200HP，每5层）
  - 🔵 **商店** - 购买卡牌/遗物（暂时跳过）
  - 🟡 **宝箱** - 获得金币

### 4. 战斗
- **手牌区**：底部显示可用卡牌
- **敌人区**：顶部显示敌人状态
- **玩家状态**：左下角（生命、护盾）
- **能量显示**：右下角
- **操作**：
  - 拖拽卡牌到敌人上 → 打出卡牌
  - 点击"结束回合"按钮 → 敌人行动

### 5. 战斗结算
- **胜利** → 显示奖励界面
- **失败** → 返回主菜单

### 6. 奖励选择
- 随机显示3张卡牌
- 点击卡牌 → 加入牌库
- 或点击"跳过" → 不拿奖励
- 自动前往下一层

### 7. 循环
地图 → 战斗 → 奖励 → 地图 → ...

## 📊 游戏系统说明

### 卡牌系统
- **总卡牌数**: 125张（超过目标120张）
- **类型**: 攻击卡/技能卡/能力卡
- **稀有度**: 普通/稀有/史诗/传说
- **效果**: 
  - 攻击卡 → 造成伤害
  - 技能卡 → 获得护盾/治疗
  - 能力卡 → 持续增益

### 战斗系统
- **回合制**: 玩家回合 → 敌人回合
- **能量**: 每回合3点，打牌消耗
- **护盾**: 回合结束清除
- **抽牌**: 回合开始抽5张

### 敌人AI
- **普通怪**: 40HP，攻击5-7
- **精英怪**: 60HP，攻击8-12
- **BOSS**: 200HP，攻击15-25
- **行为**: 攻击/防御/增益（随机）

### 地图系统
- **总层数**: 15层
- **BOSS层**: 第5/10/15层
- **节点类型**: 5种（战斗/精英/BOSS/商店/宝箱）

### 角色差异
| 属性 | 法师 | 战士 |
|------|------|------|
| 生命 | 80 | 100 |
| 能量 | 3 | 3 |
| 风格 | 魔法输出 | 反击防御 |

## 💡 游玩建议

### 新手建议
1. **先选法师** - 玩法简单，输出高
2. **优先拿攻击卡** - 快速结束战斗
3. **避开精英怪** - 前期容易死
4. **护盾很重要** - 记得拿防御卡

### 进阶策略
1. **平衡卡组** - 攻击+防御+能力
2. **关注能量** - 低费卡更灵活
3. **利用护盾** - 回合结束才清除
4. **针对敌人意图** - 看敌人下回合做什么

## 📞 遇到问题？

### 常见问题

**Q: 启动后黑屏？**  
A: 检查是否添加了GameBootstrap组件，是否点击了Play

**Q: 卡牌拖不动？**  
A: 确保场景中有EventSystem

**Q: 没有敌人？**  
A: 这是BUG，重启游戏

**Q: 战斗结束后卡住？**  
A: 手动创建BattleResultUI组件（应该自动创建）

### Debug方法
1. 打开Console窗口查看日志
2. 所有操作都有日志输出
3. 错误会以红色显示

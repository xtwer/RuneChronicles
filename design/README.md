## 🎯 设计核心

### 核心玩法
**符文融合系统：** 任意2张卡牌可融合产生新效果

### 核心循环
```
选择卡牌 → 打出/融合 → 击败敌人 → 获得奖励 → 
强化牌组 → 挑战更强敌人 → 通关/失败
```

### 核心数值
- 玩家起始生命：70
- 玩家起始能量：3
- 融合消耗：3点融合点
- 1点能量基准：6-7点数值

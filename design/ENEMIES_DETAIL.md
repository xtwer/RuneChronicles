## 设计原则

### 1. 可预测性
- 显示下回合动作（图标）
- 行为模式有规律
- 玩家可以制定策略

### 2. 多样性
- 不同攻击模式（单体/AOE/DOT）
- 不同防御机制（护盾/反击/回血）
- 不同特殊机制（召唤/增益/减益）

### 3. 难度梯度
- 层1-5：简单敌人（HP 30-60）
- 层6-10：中等敌人（HP 60-100）+ 精英
- 层11-15：困难敌人（HP 80-120）+ 精英 + BOSS

## 二、精英敌人（5种）

### 2.1 符文骑士

**定位：** 高伤害+反击

**数据：**
- 生命值：120
- 攻击力：15
- 出现层数：6-14

**行为模式：**
```
第一阶段（HP>50%）：
  回合1：攻击（15点伤害）
  回合2：双重斩击（12点伤害×2）
  回合3：防御反击（10护盾，受击反击8点）

第二阶段（HP<50%）：
  回合1：狂暴冲锋（20点伤害）
  回合2：旋风斩（对玩家造成18点伤害，忽略护盾）
  回合3：短暂恢复（回复10点生命）
```

**特殊机制：**
- 每次受到攻击有30%几率反击6点

**掉落：**
- 1张罕见/史诗卡
- 遗物（50%概率）
- 金币（80-120）

## 三、BOSS（3个）

### 3.1 堕落符文师（层5 BOSS）

**定位：** 符文魔法大师

**数据：**
- 生命值：300
- 攻击力：变化
- 阶段：3

**第一阶段（HP 300-200）：**
```
回合1：符文打击（15点伤害）
回合2：召唤符文傀儡（召唤1个符文傀儡）
回合3：符文爆发（20点伤害）
回合4：符文护盾（获得25点护盾）
循环...
```

**第二阶段（HP 199-100）：**
```
阶段转换：清除所有负面状态，获得30点护盾

回合1：符文风暴（对玩家造成18点伤害）
回合2：召唤2个符文傀儡
回合3：融合之力（吸收所有仆从，每个+20点生命+5攻击）
回合4：符文陷阱（场地放置陷阱，玩家每打出1张卡受3点伤害，持续3回合）
循环...
```

**第三阶段（HP 99-0）：**
```
阶段转换：狂暴状态，所有伤害+50%

回合1：符文毁灭（30点伤害，忽略护盾）
回合2：终极召唤（召唤3个符文傀儡）
回合3：时空崩溃（25点AOE，玩家手牌随机弃3张）
循环...
```

**特殊机制：**
- 场上每有1个仆从，攻击力+3
- 击败后，解锁新卡牌和遗物

## 四、敌人组合设计

### 4.1 组合原则
- 前排（坦克）+ 后排（输出）
- 控制型 + 爆发型
- 召唤型 + 增益型

### 4.2 常见组合

**组合1：基础（层1-3）**
```
符文傀儡 × 2
威胁度：低
```

**组合2：攻守平衡（层4-6）**
```
冰霜守卫（前排）+ 火焰精灵（后排）
威胁度：中
```

**组合3：爆发（层7-9）**
```
雷霆法师 + 暗影刺客 × 2
威胁度：高
```

**组合4：控制（层10-12）**
```
元素使者 + 腐化符文师 + 符文猎手
威胁度：极高
```

**组合5：终极（层13-14）**
```
狂暴战士 × 2 + 符文护卫
威胁度：Boss前哨战
```

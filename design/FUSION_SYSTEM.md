## 一、融合系统机制

### 1.1 融合规则
- 消耗：2张卡牌 + 3点融合点
- 生成：1张新卡牌（融合卡）
- 有效期：当前战斗
- 稀有度：提升1级（最高传说）

### 1.2 融合点获取
- 打出卡牌：+1点
- 击败敌人：+2点
- 特殊事件/遗物：额外来源
- 上限：10点

### 1.3 融合类型
1. **预设配方**：50种固定效果
2. **随机融合**：算法生成新效果

## 三、随机融合算法

### 3.1 算法规则

当玩家融合的2张卡没有预设配方时，使用随机融合算法：

**基础公式：**
```
新卡能量 = (卡1能量 + 卡2能量) / 1.5 (向上取整)
新卡效果 = 合并两张卡的主要效果
新卡强度 = (卡1强度 + 卡2强度) × 1.2
```

**示例：**
```
火焰弹（1能量，8伤害）+ 穿刺（1能量，5伤害+无视护盾）
= 烈焰穿刺（2能量，造成16点火焰伤害，无视护盾）
```

### 3.2 效果合并规则

**伤害类：**
- 同类伤害：相加后×1.2
- 不同类：保留主伤害类型，附加次要效果

**控制类：**
- 冻结+灼烧：保留两种效果
- 冻结+冻结：持续时间+1
- 灼烧+灼烧：层数+2

**特殊类：**
- 抽牌+抽牌：抽牌数+1
- 护盾+护盾：护盾值×1.3
- 抽牌+护盾：两种效果都保留，数值×0.8

## 四、融合推荐系统

### 4.1 新手推荐融合

**初期（层1-5）：**
1. 火焰弹 + 火焰弹 = 烈焰风暴（AOE清场）

1. 火球术 + 冰锥术 = 蒸汽爆炸（复合伤害）

3. 火焰风暴 + 霜冻新星 = 冰火风暴（AOE+控制）

**后期（层11-15）：**
1. 陨石术 + 冰河时代 = 世界末日（终极AOE）
2. 天罚 + 雷神之怒 = 终极天罚（BOSS克星）
3. 终极爆裂 + 完全治愈 = 重生之力（续航）

## 五、融合策略指南

### 5.1 针对不同敌人

**对抗单体高血量（精英/BOSS）：**
- 推荐：高伤单体融合
- 示例：雷霆一击 + 雷霆一击 = 雷神之锤（30伤）

**对抗多敌人（小怪群）：**
- 推荐：AOE融合
- 示例：火焰弹 + 火焰弹 = 烈焰风暴（AOE）

**对抗高攻击敌人：**
- 推荐：控制+伤害融合
- 示例：冰霜箭 + 冰锥术 = 冰川冲击波（冻结所有）

**对抗高防御敌人：**
- 推荐：破防融合
- 示例：穿刺 + 火焰弹 = 穿甲火箭（无视护盾）

## 六、融合平衡表

### 6.1 融合成本/收益分析

| 融合类型 | 融合成本 | 收益倍率 | 推荐度 |
|---------|---------|---------|--------|
| 同卡融合 | 低（同卡2张） | 1.6x | ⭐⭐⭐⭐⭐ |
| 同元素融合 | 中（需凑齐） | 1.5x | ⭐⭐⭐⭐ |
| 跨元素融合 | 高（稀有） | 1.8x | ⭐⭐⭐⭐⭐ |
| 攻击+技能 | 中（牺牲防御） | 1.4x | ⭐⭐⭐ |
| 随机融合 | 低（任意） | 1.2x | ⭐⭐ |

### 6.2 融合点价值

**3点融合点的价值等价于：**
- 打出3张卡（3回合累积）
- 击败1.5个敌人
- 约等于2-3点能量的卡牌效果

**结论：** 融合必须产生 >1.5倍收益才值得

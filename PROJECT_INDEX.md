## 🎮 核心特色

### 1. 符文融合系统 🧩
任意2张卡牌可融合产生新效果，创造独特玩法。

**示例：**
```
火焰弹（8伤害）+ 冰霜箭（6伤害+冻结）
= 冰火符文（12伤害+灼烧2回合）
```

### 2. 中国神话主题 🎨
水墨风美术 + 符文图腾，本土化设计。

### 3. 快节奏战斗 ⚡
单局 15-30 分钟，适合碎片时间。

### 4. 高重玩性 🔄
随机地图 + 数百种卡牌组合。

## 🚀 快速开始（3步）

### 步骤1：生成占位符（在Unity中）

1. 打开Unity项目
2. 菜单栏 → `Tools` → `Generate Placeholder Assets`
3. 等待生成完成（约5秒）

**生成内容**:
- 125张卡牌纯色占位图
- 2个角色纯色占位图
- 21个敌人纯色占位图
- 4个UI背景
- 5个UI图标
- 音效说明文件

### 步骤2：准备真实资源（可选）

**最快方案 - AI生成**:
1. 注册Midjourney（$10/月）
2. 使用Prompt生成卡牌图
   ```
   /imagine fantasy card game art, [卡牌名称], magic spell, 
   detailed illustration, dark fantasy style
   ```
3. 下载图片并重命名（Card_ATK_001.png）

**免费方案**:
1. 访问 [Freesound.org](https://freesound.org)
2. 搜索并下载音效
3. 访问 [Incompetech.com](https://incompetech.com)
4. 下载免费BGM

### 步骤3：替换资源

1. **准备文件** - 按命名规则重命名
   ```
   Card_ATK_001.png
   Character_Mage.png
   Enemy_001.png
   MainMenu.ogg
   card_draw.wav
   ```

2. **拖拽到Unity** - 放入对应文件夹
   ```
   拖拽 → Assets/Art/Cards/
   拖拽 → Assets/Audio/BGM/
   ```

3. **刷新Unity** - `Ctrl+R` / `Cmd+R`

4. **测试游戏** - 点击Play查看效果

## 💡 推荐工作流

### MVP阶段（当前）
**目标**: 验证玩法，占位符即可  
**行动**: 无需替换，直接玩游戏

**目标**: 初步视觉效果  
**行动**:
1. AI生成10-20张核心卡牌
2. AI生成2个角色
3. 下载2-3首免费BGM
4. 下载10个常用音效

**时间**: 1-2天

**目标**: 完整美术音效

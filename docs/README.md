## 🎯 项目愿景

**一句话描述：**  
一款融合符文融合机制的卡牌 Roguelike 游戏，玩家通过收集和组合符文，在随机生成的关卡中战斗。

**核心卖点：**
1. 🧩 **符文融合系统** - 任意2张卡牌可融合产生新效果
2. 🎨 **中国神话主题** - 水墨风美术 + 符文图腾
3. ⚡ **快节奏战斗** - 单局 15-30 分钟
4. 🔄 **高重玩性** - 随机地图 + 数百种卡牌组合

**差异化：**
- ✅ 符文融合（独特机制）

- ✅ 更快节奏（单局时间更短）

## 📅 开发周期

**总时长：** 12-16 周

### 阶段1：原型开发（3周）
- 核心战斗系统
- 30 张基础卡牌
- 5 个敌人
- 简单地图

### 阶段2：内容扩充（4周）
- 100 张卡牌
- 2 个角色
- 15 个敌人
- 完整地图系统

### 阶段3：美术和音效（3周）
- AI 生成卡面
- UI 美化
- 基础音效/音乐

### 阶段4：打磨发布（2-4周）
- 平衡性调整
- Bug 修复
- Steam 页面准备
- Early Access 上线

## ⚠️ 风险评估

### 技术风险（低）

- **应对：** 降低范围，专注核心玩法

- **应对：** 灵活调整范围，Early Access 尽早上线

## 1. 游戏概述

### 1.1 游戏类型
**卡牌构筑 + Roguelike**

### 1.2 游戏背景
在远古时代，世界由符文之力主宰。符文师通过掌握和组合符文，获得强大的力量。玩家扮演一名年轻的符文师，在充满危险的符文塔中冒险，寻找失落的至高符文。

### 1.3 核心玩法循环
```
选择角色 → 构筑初始牌组 → 
进入关卡 → 战斗/事件/商店 → 获得奖励 → 
升级牌组 → 下一关卡 → 
BOSS战 → 通关或死亡 → 
解锁新内容 → 重新开始
```

### 1.4 核心创新点
**符文融合系统：**
- 任意2张卡牌可以融合
- 融合产生新的效果（不是简单相加）
- 融合卡牌消耗"融合点"（战斗中积累）
- 融合卡牌在当局有效，下局重置

**示例：**
```
火焰弹（造成8点伤害）+ 冰霜箭（造成6点伤害+冻结1回合）
= 冰火符文（造成12点伤害+灼烧2回合）
```

## 2. 核心玩法

### 2.1 战斗系统

**回合制战斗：**
- 玩家回合 → 敌人回合 → 循环
- 每回合抽5张卡（可调整）
- 能量系统：每回合3点能量（可增长）
- 手牌上限：10张

**行动流程：**
1. 回合开始：抽牌、恢复能量
2. 玩家阶段：打出卡牌、使用能力
3. 回合结束：弃牌、触发效果
4. 敌人阶段：敌人行动
5. 重复

**胜利条件：**
- 敌人生命值归零

**失败条件：**
- 玩家生命值归零

### 2.2 符文融合机制

**融合规则：**
- 消耗2张卡牌 + 融合点
- 生成1张新卡牌
- 新卡牌效果根据融合配方决定
- 融合卡牌稀有度提升

**融合点（Fusion Points）：**
- 初始：0点
- 获得方式：
  - 打出卡牌 +1
  - 击败敌人 +2
  - 特殊事件
- 消耗：每次融合 -3点
- 上限：10点

**融合配方：**
- 预设配方：100+ 种（有明确效果）
- 随机融合：产生意外效果

### 2.3 地图探索

**地图结构：**
```
起点 → 层1（3个节点）→ 层2（4个节点）→ ... → BOSS
```

**节点类型：**
- 🗡️ 战斗：普通敌人
- 👹 精英战斗：更强敌人 + 更好奖励
- 💰 商店：购买卡牌/遗物/服务
- 🎁 宝箱：随机奖励
- ❓ 事件：随机事件（风险/收益）
- 🔥 篝火：回血或升级卡牌
- 👑 BOSS：关底BOSS

## 3. 游戏系统

### 3.1 卡牌系统

**卡牌属性：**
- 名称
- 类型（攻击/技能/能力）
- 稀有度（普通/罕见/史诗/传说）
- 能量消耗
- 效果描述
- 可升级次数

**卡牌类型：**

1. **攻击卡（Attack）**
   - 造成伤害
   - 可附带额外效果

2. **技能卡（Skill）**
   - 获得护盾
   - 抽牌/回能
   - 控制效果

3. **能力卡（Power）**
   - 持续效果
   - 改变战斗规则

4. **融合卡（Fusion）**
   - 通过融合生成
   - 效果更强
   - 单次战斗有效

**卡牌升级：**
- 方式1：篝火升级（免费，限次数）
- 方式2：商店升级（花费金币）
- 效果：伤害+2、护盾+3、能量-1等

### 3.2 遗物系统

**遗物（Relic）：**
- 永久效果
- 每局最多携带20个
- 可以叠加

**遗物类型：**
- 🔴 攻击型：增加伤害、暴击
- 🔵 防御型：增加护盾、回血
- 🟢 辅助型：抽牌、回能、特殊机制
- 🟡 特殊型：改变游戏规则

**遗物示例：**
```
【符文之眼】
每场战斗开始时，获得1点融合点

【火焰宝石】
你的火焰伤害+50%

【生命护符】
每击败1个敌人，回复5点生命值
```

### 3.3 角色系统

**角色特性：**
- 初始生命值
- 初始卡组（10张）
- 初始遗物（1个）
- 专属机制

**计划角色数：**
- 阶段1：1个角色（符文法师）
- 阶段2：2个角色（+符文战士）
- 正式版：3-4个角色

### 3.4 进度系统

**解锁系统：**
- 通关次数 → 解锁新角色
- 击败BOSS → 解锁新卡牌
- 完成成就 → 解锁新遗物

**难度系统：**
- 基础难度（A0）
- 上升难度（A1-A20）
- 每级增加敌人强度

## 8. 数值平衡

### 8.1 核心数值

**玩家：**
- 基础生命值：70
- 最大生命值：150（通过遗物提升）
- 初始能量：3
- 最大能量：6

**敌人：**
- 普通敌人生命值：30-60
- 精英敌人生命值：100-150
- BOSS生命值：250-400

**卡牌：**
- 攻击卡：伤害/能量比 = 6-7
- 技能卡：护盾/能量比 = 5-6
- 能力卡：效果价值约等于2倍能量

### 8.2 平衡原则

1. **能量效率：** 1点能量 ≈ 6-7点数值
2. **稀有度差距：** 罕见卡牌比普通强30%
3. **融合收益：** 融合卡牌效果 > 2张卡牌总和的120%
4. **成长曲线：** 玩家强度随关卡线性增长

## 9. UI/UX设计

### 9.1 UI布局

**战斗界面：**
```
[敌人区域 - 顶部]
   敌人1  敌人2  敌人3
   HP/意图

[中间区域]
   战斗日志 / 特效

[玩家区域 - 底部]
   HP: ❤️70/70  能量: ⚡3/3  融合点: 🔹5/10
   [手牌区域 - 5张卡牌]
   [抽牌堆]  [弃牌堆]  [结束回合]
```

**地图界面：**
```
[顶部状态栏]
   生命值 | 金币 | 层数

[地图节点]
   ○ → ○ → ○
     ↓   ↘ ↓
   ○ → ○ → ○

[底部菜单]
   [牌组] [遗物] [设置]
```

### 9.2 交互设计

**拖拽打牌：**
- 拖动卡牌到敌人 = 攻击
- 拖动卡牌到自己 = 使用
- 松手取消

**融合操作：**
- 长按卡牌A → 拖动到卡牌B → 显示融合预览 → 确认

**快捷操作：**
- 双击卡牌 = 快速使用
- 右键 = 查看详情
- 空格 = 结束回合

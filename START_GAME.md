# 🎮 如何启动游戏

**最后更新**: 2026-03-23 17:46

---

## ✅ 快速启动（3步）

### 步骤1：打开Unity项目
```bash
cd ~/Desktop/RuneChronicles/RuneChronicles
```

然后在Unity Hub中打开这个项目，或者直接双击 `RuneChronicles.unity` 项目文件。

### 步骤2：打开GameStart场景

在Unity中：
1. **Project窗口** → 找到 `Assets/Scenes/GameStart.unity`
2. **双击打开**

### 步骤3：点击Play
点击Unity顶部的 **Play按钮** ▶

**完成！** 游戏会自动：
- 初始化所有系统（13个Manager）
- 显示主菜单
- 你就可以开始玩了！

## 🎯 游戏流程

### 1. 主菜单
- **开始游戏** → 进入角色选择
- **设置** （未实装）
- **退出游戏**

### 2. 角色选择
- **符文法师** - 生命80，魔法输出
- **符文战士** - 生命100，反击防御

点击角色卡片选择

### 3. 地图探索
每层3个节点可选：
- 🟢 **战斗** - 普通敌人（40HP）
- 🔴 **精英** - 强力敌人（60HP）
- 🔥 **BOSS** - 超强敌人（200HP，第5/10/15层）
- 🔵 **商店** - 购买卡牌/遗物
- 🟡 **宝箱** - 获得金币
- 🟣 **融合** - 点击右上角"融合"按钮

### 4. 战斗
- **拖拽卡牌** - 拖到敌人上打出
- **结束回合** - 点击右下角按钮
- **查看状态** - 左下角（生命/护盾），右下角（能量/回合）

### 5. 战斗结算
- **胜利** → 选择奖励卡牌
- **失败** → 返回主菜单

### 6. 奖励选择
- 随机3张卡牌
- 点击选择，加入牌库
- 或点击"跳过"

### 7. 商店（可选）
- **买卡牌** - 50金币/张
- **买遗物** - 150金币/个
- **删卡牌** - 75金币/次

### 8. 融合（可选）
- 选择2张卡牌
- 消耗融合点
- 获得新卡牌

## 🎮 操作方式

### 战斗中
- **打出卡牌**: 鼠标拖拽卡牌到敌人上
- **结束回合**: 点击右下角"结束回合"按钮
- **查看状态**: 
  - 左下角 = 玩家生命/护盾
  - 右下角 = 能量/回合数
  - 顶部 = 敌人状态

### 地图中
- **选择路径**: 点击节点按钮
- **进入融合**: 点击右上角"融合"按钮
- **节点颜色**:
  - 绿色 = 普通战斗
  - 暗红色 = 精英战斗
  - 鲜红色 = BOSS战
  - 蓝色 = 商店
  - 金色 = 宝箱

## 💡 新手建议

### 第一次玩
1. **选法师** - 玩法简单，输出高
2. **避开精英** - 前期容易死
3. **优先拿攻击卡** - 快速结束战斗
4. **记得用护盾** - 回合结束才清除

### 进阶策略
1. **平衡卡组** - 攻击+防御+能力
2. **关注能量** - 低费卡更灵活
3. **商店优先买遗物** - 遗物永久生效
4. **善用融合** - 传说卡非常强

## 🐛 常见问题

### Q: 启动后黑屏？
**A**: 确保选中了 `GameStart.unity` 场景，然后点击Play

### Q: 没有主菜单显示？
**A**: 
1. 检查Console是否有错误（红色日志）
2. 确保GameBootstrap组件正确添加
3. 尝试 `Cmd+R` 刷新Unity

### Q: 卡牌拖不动？
**A**: 
1. 确保场景中有EventSystem
2. 如果没有，GameBootstrap会自动创建

### Q: 编译错误？
**A**: 
1. 打开Console窗口查看错误
2. 尝试 `Assets → Reimport All`
3. 或通知我修复
